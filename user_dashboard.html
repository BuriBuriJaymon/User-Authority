How It Works

1. Report an Issue

Snap a photo, add details, and submit your report. It's saved right here in your browser.

2. Track Status

Go to 'My Reports' to track your submissions. An 'Authority' using *this same browser* can see and update the status.

3. Reopen if Needed

If an issue is marked 'Resolved' but the problem isn't fixed, you can reopen the report from the 'My Reports' page.
